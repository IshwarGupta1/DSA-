DevQA Endpoints
POST /api/DevQA/{gameCode}/join
Join a game by its gameCode.
Path Parameter:

gameCode: The unique identifier for the game.
Response:
200 OK on success.
400 Bad Request or 404 Not Found on error.
POST /api/DevQA/{gameCode}/vote
Vote for a game by its gameCode.
Path Parameter:

gameCode: The unique identifier for the game.
Request Body:
json
Copy code
{
  "vote": "string"
}
Response:

200 OK on success.
400 Bad Request on error.
GET /api/DevQA/{gameCode}/devvotes
Get all developer votes for a game.
Path Parameter:

gameCode: The unique identifier for the game.
Response:
200 OK with votes data.
GET /api/DevQA/{gameCode}/qavotes
Get all QA votes for a game.
Path Parameter:

gameCode: The unique identifier for the game.
Response:
200 OK with QA votes data.
ScrumMaster Endpoints
POST /api/ScrumMaster/create
Create a new Scrum game.
Response:

201 Created on success.
POST /api/ScrumMaster/{gameCode}/reveal
Reveal the votes for a specific game.
Path Parameter:

gameCode: The unique identifier for the game.
Response:
200 OK on success.
401 Unauthorized or 404 Not Found on error.

How to Use the API
Register a new user by sending a POST request to /api/Auth/register.
Login using the /api/Auth/login endpoint.
Create a game by sending a POST request to /api/ScrumMaster/create.
Join and vote on a game using the /api/DevQA/{gameCode}/join and /api/DevQA/{gameCode}/vote endpoints.
Check votes using the /api/DevQA/{gameCode}/devvotes and /api/DevQA/{gameCode}/qavotes endpoints.
Reveal the votes for a game using the /api/ScrumMaster/{gameCode}/reveal endpoint.
Feel free to explore the API and contribute to the ScrumPoker project!
